
# 🎉 Event Management Website

## 📌 Description
This is a web-based event management system where users can view events and organizers can create and manage events.

## 🚀 Features
- Create events
- Book tickets
- Organizer dashboard
- Event listings

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
